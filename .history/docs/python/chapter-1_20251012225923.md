## 身份证号的性别判断

身份证号顺序码第三位,偶-女性,奇-男性

**I(imput)  P(processing)  O(output)**


# 身份证号码的判断程序（这是一个注释）
id_code=input("请输入18位身份证号码:")
number=int(id_code[-2])
if	number%2==0:
    print("女")
else:
    print("男")


> `input` 强制转化字符串
> `int` 将字符串转化为数字 从而进行数值运算
> `str`代表字符串
>
> =意味着赋值,== 意味着判断是否等于
> if else:**分支语句**
>
> 字符串要用引号引用

另强调,使用英文输入法的标点;拼写注意大小写
