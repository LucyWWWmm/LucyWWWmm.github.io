# 程序设计基本概念

1. 计算机：根据指令操作数据的一种设备，人与计算机交流的过程就是编程

2. 程序设计语言：是一组记号和一组规则

3. 语言的发展：第一代机器语言，第二代汇编语言，第三代高级语言（一、二代叫低级语言）

	高级语言分为静态语言（编译执行，如C语言）与动态语言（解释执行，如Python）

4. 编译执行：一次性翻译，一旦程序被编译，不再需要源代码脱离母体直接执行

	解释执行：无法分离源代码，带有解释器

# python语法基本要素

1. 变量命名规则：数字不能开头

2. 关键字35个，大小写敏感，如“True“

	基本数据类型：数值，字符串，布尔类型，空类型

	组合数据类型：集合，列表，元组，字典

	集合，字典的成员必须是 **不可变数据类型** （数值，字符串，不含可变数据类型的元组）

	集合字典无序，列表元组有序，这四个都是可遍历结构

	---

	运算符：算数，赋值，关系（>,<），成员（in，not in），逻辑（not，and，or）

	优先级：括号>幂运算>正负号>算数>关系>逻辑>赋值
