请用户随机输入一行字符，用程序分别统计出其中字母（包含大写字母和小写字母）、空格、数字和其它字符的个数。

temp = input("请输入一个字符串：")
a=0
b=0
d=0
e=0
for c in temp:
    if c.isalpha():
        a +=1
    elif c.isspace():
        b +=1
    elif c.isdigit():
        d +=1
    else:
        e +=1

print(f"该字符串中字母数目为： {a}")
print(f"该字符串中数字数目为： {d}")
print(f"该字符串中空格数目为： {b}")
print(f"该字符串中其他字符数目为： {e}")

注： += 表示加后面的数字再带入回去
